# Simulation-GPT • Privacy Notice

**Who we are**  
Simulation-GPT is an automation assistant that creates and manages physics-experiment repositories in the organisation **PhysicsSimLAB**.

---

## 1 What data do we collect?

| Type of data | Where it comes from | Where it is stored |
|--------------|--------------------|--------------------|
| Source code, configuration files, simulation results that **you send in chat** | Your prompts to Simulation-GPT | The specific experiment repository created for you (private by default) |
| Minimal technical logs (endpoint success/failure, timestamp) | Backend server | Ephemeral in memory; **no code or personal info** is logged |

---

## 2 How do we use the data?

* To version-control your experiment and let collaborators work on it.  
* To run automated actions you request (e.g. “run the simulation”, “commit and push”).  
* We **never** analyse, mine, or resell your code or data.

---

## 3 Retention & deletion

* **Delete the repo ⇒ all code and artefacts are gone immediately** from the organisation.  
* Backend logs self-purge after 30 days.

---

## 4 Third-party sharing

* Your data is kept entirely within GitHub infrastructure.  
* We do **not** transmit it to any other service or analytics provider.
